The arcs for the angle benchmark are generated by

```bash
./GCR_Angle_test/GCR_generate_arcs.cpp
```

To compile, from the project root

```bash
cd ./GCA_Angle_test

# adjust compiler / flags if needed
g++ -O3 -std=c++17 -I.. generate_arcs.cpp -o generate_arcs

```

To run the executable

```bash
./GCR_generate_arcs [seed] [total] [ill]
- seed — RNG seed (default: 42)

- total — total number of arcs to generate (default: 10000000)

- ill — number of ill-conditioned arcs to sprinkle in (default: 100000)
```

the `results` directory is created automatically if it does not exist.
